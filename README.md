# E-commerce Analytics: Когортный и RFM-анализ маркетплейса Olist
Проект посвящен комплексному анализу клиентской базы бразильского маркетплейса **Olist** (данные за 2017-2018 гг.). 

Цель – оценить удержание клиентов, выявить паттерны поведения и сегментировать аудиторию для оптимизации маркетинга.

Исходный датасет взят с платформы Kaggle: [(https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce)].
## Стек
* **Python** (Pandas, NumPy, Matplotlib, Seaborn, Statsmodels)
* **Методы:** Когортный анализ (Retention Rate), RFM-сегментация, проверка статистических гипотез.
## Проверяемые гипотезы
* **H1:** Так как Olist – non-FMCG платформа, Retention Rate 1-го месяца не превысит 1.5% ни в одной когорте.
* **H2:** Осенние когорты (подготовка к распродажам) имеют удержание на 1-м месяце значимо выше, чем весенне-летние.
* **H3:** На 12-м месяце жизни когорт будет локальный всплеск удержания из-за ежегодной сезонности.
* **H4:** Узкая группа VIP-клиентов генерирует непропорционально большую долю выручки платформы.
## Ключевые результаты анализа
* **Удержание (H1 и H3):** * Гипотеза **H1 подтвердилась** – Retention Rate 1-го месяца стабильно держится ниже 1% (в среднем 0.32-0.74%), что доказывает ультра-низкую частоту повторных покупок (повторные покупки совершают менее 3% всей базы). Гипотеза **H3 подтвердилась статистически, но бизнесово нерелевантна**. На 12-м месяце жизни зафиксирован локальный пик удержания, однако объём вернувшихся клиентов (n < 20) критически мал для принятия операционных решений. Паттерн требует верификации на данных за 2-3 года и анализа категорий товаров (`order_items`).
* **Сезонность (H2):** Гипотеза **H2 частично подтвердилась** (на уровне тенденции). Тест Манна-Уитни зафиксировал статистически значимое различие (p < 0.05), однако из-за ограничений выборки (n=2 для осенних когорт) мощность теста крайне низкая. Результат указывает на позитивный тренд в период осенних распродаж, но требует верификации на данных за 3-4 года.
* **RFM-сегментация (H4):** Гипотеза **H4 подтвердилась**. Выявлен ключевой сегмент – **«Перспективные с высоким чеком»** (недавние клиенты с высоким чеком, ещё не ушедшие в отток). Они составляют всего **16.5% базы, но генерируют 30.4% всей выручки**. При этом подавляющую часть клиентской базы платформы составляют сегменты с однократными покупками в глубоком оттоке (**«Одноразовые / Малоценные»** и **«Уснувшие крупные клиенты»**).

**Стратегические приоритеты:**
  1. Внедрение **Welcome-цепочек для «Перспективных»** – оффер на вторую покупку в течение 14-30 дней, чтобы не дать им уйти в отток.
  2. Реактивация **«Уснувших крупных клиентов»** через таргетированные рассылки по категории первой покупки.
  3. Удержание **«Чемпионов»** и **«Уснувших VIP»** с помощью программ лояльности.
